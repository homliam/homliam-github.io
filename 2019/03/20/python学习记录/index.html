  • 格式化操作符
  • 如果希望输出文字信息的同时，一起输出数据，就需要用到格式化操作符

    • 包含%的字符串，被称为格式化字符串

%f 浮点数，%.02f表示小数点后只显示两位

1.2 逻辑运算符

  • 在程序开发中，通常在判断条件时，会需要同时判断多个条件

  • 只有多个条件都满足，才能执行后续代码，这个时候就需要使用到逻辑运算符

  • 逻辑运算符可以把多个条件按照逻辑链接，形成更复杂的条件

  • python中的逻辑符包括：与and/或or/非not 三种

  • and 要求两个必须同时满足为True，or要求有一个满足即为True，都是连接两个条件

  • not用来用来颠倒条件。原条件成立结果变为不成立，原条件不成立结果变为成立

  • 在开发中，通常希望某个条件不满足是，执行一些代码，可以使用not

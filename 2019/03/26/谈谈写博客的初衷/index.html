谈谈写博客的初衷

发表于 2019-03-26
字数统计: 1.3k 字 | 阅读时长 ≈ 4 分钟 


        

        

记录带来积累，自律带来自由




在12年高中毕业之后，在去年之前，我已经是一个与写作越走越远的人了。没有了考试的压力之后，这项曾经的制胜法宝般的技能就像蒙尘的玛瑙，不再光彩。


直到去年开始接触编程，接触如今炙手可热的互联网的底层，我看到了互联网技术上特有的闪光点，让我猛然激起了写博客的欲望。


编程影响我写博客的有三个点：


    

  • 注释
    • 

  • 开源
    • 

  • README
    • 



作为精装修行业从业人员，和编程本来是八杆子打不着的，我应该专注的是设计、审美、施工做法、项目管理。但是偶然看到了一片广告软文，大概讲的就是：10后已经是互联网的原住民，python已经加入了小学课本，通过编程达到训练逻辑能力已经刻不容缓，人工智能已经是大势所趋，高薪工作虚位以待云云。然而这样的打鸡血广告怎么可能吸引到我呢，直到看到最角落里的一句话“Coding Changes the World”，代码改变世界，嗯哼，有点意思。




抱着改变世界的心态，我安装的Python3.7，下载了最全面的包管理工具 Anaconda，安装了最强大的编译器 Pycharm,最优秀的文本工具 Sublime 终于开始了学习之路，
“工欲善其事，必先利其器”，我安装了最受欢迎的发行版本Linux虚拟系统Ubuntu，终于可以快乐的面向对象编程了。




可是改变世界并没有那么简单，有 debug 就必然会有 bug ，有 bug 就得 Google ，然后问题就得以解决。就在不断得制造问题，解决问题得过程中，又一件事情让我感到惊讶，无论我碰到什么问题，网上总是能找到详尽得解决方案，极简的排版，细致的分析，每一份答案就像是班上成绩最好的那个学生写的错题本。可是在我自己的行业里想找一点资料确是难上加难，想要查一篇规范，网上看似泛滥的资源，要挑到有用的部分确实大海捞针。想找一份做法图纸，不想花钱的话基本上是不可能。想找一些材料或者是贴图素材，全部都被收费网站垄断，花钱下载下来都不一定是自己想要的。


以前我在新世界打开了一扇大门，进了吾爱破解论坛,从此资源如我是浮云，工作效率杠杠提升，实用工具数不胜数。现在我又打开了一扇新的大门
github,但是一开始最大的收益是英语水平明显提升，然后才是开源。


慢慢的，我知道了那些极简的排版是 MRAKDOWN 格式写出来的，那些细致的分析不光是技术分享，更是自己的技术积累。也是接触了这些之后，我才知道早就下载好的Typora是怎么用的，以前都是把它当作一个高级的记事本来用，现在把Notepad++当作高级记事本来用。


扯了这么多废话，其实我的初衷就是：记录和自律。


记录
在工作中使用软件经常会有这样的场景，一个 CAD 问题前后百度了几次，用一次，查一次；一个 PS 参数和技巧也是如此；一个Sktchup好用的操作方法甚至网上很难才能查到，结果学会了又被自己错过。本来就是空余的零散时间来学习编程，很多问题也是经常会忘。所以我的初衷就是做一些记录，这是一个积累的过程，也是一个分享的过程，以前不理解白白分享的意义何在，现在明白，一个人在工作岗位上所能发挥出来的价值是有限的，在互联网这个大平台上发挥的价值确实无限的，能帮到别人便是价值的体
现，有价值必然就有收获。这便是写博客记录的意义。


自律
“自律使人自由” 坚持做有意义的事情才是自律，才能使人自由，坚持打麻将就不算。


我觉得写作就是意见很有意义的事情，这么多年都太缺乏思考了，丧失了太多的表达能里，叙述能力，逻辑能力。写作不同于演讲，需要那么强的应变能力，它有足够的时间 去积累和沉淀，说过的话很快就能忘了，但是写下来的字可以一遍又一遍的看。自律一时爽，一直自律一直爽。


多年不提笔，不构思，寥寥几行字，却很明显的感受到了力不从心，词不达意。 希望一段时间的坚持可以改变一点现状。


自勉。
